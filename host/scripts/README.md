Midi Sequencer
===============
Requires: 
* python-midi 
  * https://github.com/vishnubob/python-midi
* pyserial
  * brew install pyserial

Installation:
* Follow installation instructions in python-midi README

Midi Video
===============
Requires: pygame

Installation (OSX):
```
$ brew install pip
$ brew install portmidi
$ pip install hg+http://bitbucket.org/pygame/pygame
```
